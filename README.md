# RuneguardTactics
A C++ 20 &amp; Raylib 3.0+ Tactics game built with minimal AI usage to write code. Used by creator for learning and eventual commercial release. 

## AI Usage
AI use is explicitly used as a glorified google search, to assist with git issues (still very new to git) and other questions I might have about how something works. I try to adhere as best I can to a 90/10 percentage of Human/AI writing, reading, and debugging the codebase. 

Any commits with AI written code will have that mentioned in its commit message. I will add a reference in this main page with references to where AI assisted code was used, with at minimum monthly updates.

As the community grows with Runeguard Tactics, any assistance with bugs, features, etc. is to be given freely, without compensation or reimbursement. That being said, your assistance is much preferred over using AI. We need to stick together as a people and do what we can to unify ourselves, even in small ways. So if you ever provide assistance, whether it helped or not, it is appreciated.

## Installing Rune Guard Tactics to test yourself
It is always recommended to clone from main. You will need to download raylib here: https://www.raylib.com/ use the itch.io download for ease of use. Create a parent directory on your machine, unzip raylib to that parent directory. Clone Rune Guard Tactics in that Parent Directory, not the raylib directory. cmake should be able to find the raylib directory from the parent directory.
