# RuneguardTactics
A C++ 20 &amp; Raylib 3.0+ Tactics game built without using AI to write any code. Used by creator for learning and eventual commercial release. AI use is explicitly only used for syntax, library, and other questions the developer may have, however, the community is preferred over AI in this regard. Any assistance with bugs, features, etc. is to be given freely, without compensation or reimbursement.
